## 01 - Introdução ao comando "SHOW DATABASES"

> O comando **"SHOW DATABASES"** é responsável por exibir os banco de dados existentes em nassa base de dados.

Por exemplo:

```sql
SHOW DATABASES;
```

**OUTPUT:**  
```sql
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+
4 rows in set (0.02 sec)
```
