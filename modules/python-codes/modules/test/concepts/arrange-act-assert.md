## Intro to Arrange, Act and Assert (AAA)

Most functional tests follow the **Arrange-Act-Assert model**:

 - **Arrange:**
   - Or set up, the conditions for the test.
 - **Act:**
   - By calling some function or method.
 - **Assert:**
   - That some end condition is true.
